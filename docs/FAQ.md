# FAQ

## What is this repository?

It is an ImagineVid-maintained collection for source-verifiable Seedance 2.5 prompts, playable video results, and production notes. Every published case must retain its creator, canonical source, explicit model evidence, and playable result.

## Is Seedance 2.5 publicly documented?

Reporting in July 2026 identifies Seedance 2.5 as ByteDance's next Seedance video release. ByteDance's public Seed model catalog does not yet provide a complete Seedance 2.5 model card, stable public API identifier, or detailed specification. The repository therefore avoids repeating numeric capability claims until a durable first-party source supports them.

Current status references:

- [Seedance 2.5 release reporting](https://www.theinformation.com/briefings/bytedance-unveils-seedance-2-5-video-model)
- [ByteDance Seed model catalog](https://seed.bytedance.com/en/models)

## Why create the collection now?

Early model releases produce a noisy mix of first-party demos, reposts, mislabeled clips, and prompt fragments. Establishing the evidence rules before importing cases prevents launch excitement from becoming permanent bad data.

## Where does the ImagineVid link go?

Until ImagineVid publishes a dedicated Seedance 2.5 route, repository calls to action use the existing [Seedance 2.0 workflow](https://imaginevid.io/seedance-2-0). The label makes this fallback explicit rather than pretending the older route is a 2.5 product page.

## How are prompts selected?

A community case enters only after source verification, explicit Seedance 2.5 model confirmation, playable-video review, workflow classification, and duplicate checks. Launch announcements, disconnected still images, and prompts without attributable results are excluded.

## What makes a video prompt reusable?

A reusable brief names visible action, gives the camera a path, orders the important beats, assigns roles to references, specifies sound where relevant, and states what must remain stable. It teaches a production decision, not merely an aesthetic mood.

## When will more languages be added?

English is the launch truth source. Additional languages will be generated only after their editorial copy and prompt overlays have been reviewed independently; copied translations from sibling repositories are not accepted.
