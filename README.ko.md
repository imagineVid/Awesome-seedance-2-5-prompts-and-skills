## Seedance 2.5란?

**Seedance 2.5**는 2026년 7월 보도를 통해 알려진 차세대 Seedance 비디오 모델의 명칭입니다. ByteDance의 공개 Seed 모델 카탈로그에는 아직 완전한 Seedance 2.5 모델 카드, 안정적인 API 식별자 또는 상세 공개 사양이 없습니다. 따라서 이 저장소는 확인된 1차 자료와 커뮤니티 주장을 구분하며, ByteDance가 지속적으로 참조 가능한 문서를 공개하면 범위를 갱신합니다.

프롬프트 라이브러리는 실제 결과에서 확인할 수 있는 요소에 집중합니다. 참조 자료의 역할, 화면에 보이는 동작, 카메라 경로, 비트 타이밍, 대사와 사운드 큐, 연속성 제약, 편집 의도를 다룹니다. 수치형 성능 주장은 안정적인 1차 출처로 확인되기 전까지 제외합니다.

- **각 참조 자료에 역할 지정** - 어떤 입력이 정체성, 구도, 움직임, 사운드 또는 시각 처리를 제어하는지 명시합니다
- **보이는 원인과 결과 설명** - 각 행동을 화면에 나타날 움직임, 반응 또는 환경 변화와 연결합니다
- **장면을 비트로 구성** - 줄거리 요약 대신 관찰 가능한 순간을 순서대로 나눕니다
- **사운드트랙 연출** - 장면을 진행시키는 지점에만 대사, 폴리, 환경음과 음악을 지정합니다
- **카메라 경로 명시** - 프레이밍, 이동, 피사체 거리와 주요 샷 사이의 전환을 정의합니다
- **연속성을 제약으로 작성** - 시퀀스 전반에서 인물, 의상, 제품 형태, 공간 배치와 조명을 보호합니다

**현재 참고 자료:** [Seedance 2.5 release reporting](https://www.theinformation.com/briefings/bytedance-unveils-seedance-2-5-video-model) · [ByteDance Seed model catalog](https://seed.bytedance.com/en/models) · [Available Seedance workflow on ImagineVid](https://imaginevid.io/seedance-2-0)

### 프롬프트를 샷 템플릿으로 바꾸기

재사용 가능한 비디오 프롬프트는 장면 변수와 연출 논리를 분리합니다. 검증된 카메라 경로, 비트 구조, 사운드 계획, 연속성 규칙은 유지하면서 주제, 배경, 대사 또는 제품을 바꾸세요.

**템플릿 패턴:**
```
[DURATION + ASPECT RATIO]. [SUBJECT]가 [SETTING]에서 [VISIBLE ACTION]을 수행한다. Camera: [FRAMING + MOVE]. Beats: [TIMED ACTIONS]. Audio: [DIALOGUE + FOLEY + AMBIENCE]. Preserve: [IDENTITY / PRODUCT / LAYOUT]. Avoid: [FAILURE MODES].
```

동작 하나와 카메라 아이디어 하나로 시작하세요. 눈에 보이는 제작 요구를 해결할 때만 타이밍, 오디오, 보존 제약을 추가하고, 생성 간에는 한 번에 변수 하나만 바꾸세요.
